Tambem aprendemos a usar as tags de formatação

  • strong = Negrito no texto
  • em = ênfase no texto
  • mark = Para marcar um tento como se fosse uma caneta de marcação
  • Small = para textos maiores
  • del = Para riscar o texto e dizer que ele deve ser ignorado
  • ins = Para mostrar que o texto é novo ou deve ser adiconado
  • sup = Para textos menores acima do texto principal, muito usado para fórmulas matemáticas
  • sub = Texto menor embaixo, para fórmulas.
