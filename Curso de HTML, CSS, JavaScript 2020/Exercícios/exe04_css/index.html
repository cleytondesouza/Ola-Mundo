Como Criar um Arquivo em CSS?

Para criar um arquivo em CSS basta usar a extensão .css

Como vincular uma página ao arquivo CSS?

Para vincular um aquivo CSS basta fazer como no seguinte exemplo:
link rel="stylesheet" href="style.css"
Coloque esse link entre tags HTML

Outras Formas de usar o CSS

Forma 01

Você pode usar dentro das tags, com o parâmetro 'style' e os valores colocados serão as formatações como na página sepada em CSS

Forma 02

Você pode usar na área de head antes do body em nossa página html

Para isso basta usar a tag style e fecha-la também, e entre essa tags coloque as formatações

Classes em CSS

Para usar as classes, dentro do ser aquivo css coloque um ponto(.) + o nome da sua classe

Depois para usar as configurações dessa classe, basta usar o parâmetro class e colocar o valor com o nome da classe
